## Tecnologias e Práticas Adotadas
  
  Para este projeto, exercitamos os princípios da arquitetura SOLID e os princípios de POO em uma estrutura de jogos de interpretação de papéis, mais conhecidos como jogos RPG (Role Playing Game) utilizando o TypeScript.

  
  


## Orientações para iniciar o projeto

Clone o projeto e acesse a pasta do mesmo.

$ git clone git@github.com:Cristks/trybers-and-dragons.git

$ cd sd-024-b-project-trybers-and-dragons

<details>
  <summary><strong>🐋 Rodando no Docker vs Localmente</strong></summary><br />
  
  <details>
  <summary><strong>Com Docker</strong></summary>

  ```bash
  # Criar container
  $ docker-compose up -d

  # Abrir terminal interativo do container
  $ docker exec -it trybers_and_dragons bash

  # Instalar as dependências
  $ npm install

  # Iniciar o projeto
  $ npm start
  ```
</details>

<details>
  <summary><strong>Sem Docker</strong></summary>

  ```bash
  # Instalar as dependências
  $ npm install

  # Iniciar o projeto
  $ npm start
  ```
</details>
